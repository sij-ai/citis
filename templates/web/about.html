About Citis

Permanent web archiving and citation for researchers, journalists, and professionals

Permanent Preservation

Your archived content is stored permanently with multiple backup systems. Unlike traditional bookmarks or links, Citis archives never break or disappear.

Instant Archiving

Create full-page archives in seconds using SingleFile and ArchiveBox integration. Preserve styling, images, and interactive elements exactly as they appear.

Text Fragment Highlighting

Highlight specific text passages within archived content for precise citation. Perfect for academic research and professional documentation.

Detailed Analytics

Track access patterns, geographic distribution, and usage statistics. Understand how your archived content is being accessed and used.

How It Works

Submit URL

Paste any web page URL into our archiving system via the web interface or API.

Automatic Capture

Our system captures the full page content, including CSS, JavaScript, and media files.

Permanent Link

Get a permanent shortcode link that will always work, perfect for citations and references.

Technology Stack

Archiving Engines
  • SingleFile: High-fidelity page preservation
  • ArchiveBox: Multi-format archive generation
  • PDF Generation: Portable document format support
Platform Features
  • REST API: Full programmatic access
  • Django Framework: Robust and scalable backend
  • Bootstrap UI: Modern, responsive interface
